# Article édité

Les React Hooks ont révolutionné la façon dont nous développons des composants React. 
Dans ce guide, nous explorons les hooks essentiels et leurs cas d'usage.

## useState : Gérer l'état local

Le hook `useState` permet de gérer l'état dans les composants fonctionnels. 
C'est l'un des hooks les plus utilisés.

```jsx
const [count, setCount] = useState(0)
```
